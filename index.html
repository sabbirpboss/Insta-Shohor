Q3: What is different of Local Storage and Session Storage?

Answer-Q3: Local storage stored data in web browsers. It save the user information permanently in the browsers. And session storage also stored data in web browsers. But it is not permanent. After closing the browsers window it cleard the saved data. Session storage exits only within the current window. But Local storage do not do that. Local storage has no expiry date. It will be available for the users after closing the window. Local storage uses in various types. Shopping cart, user login information etc. Data stored in session storage is specific to the protocol of the page.Local storage data is specific to the protocol of the document. Local storage data for a document loaded in a "private browsing" or "incognito" session is cleared when the last "private" tab is closed.
